# NKT — прототип страницы каталога

Интерактивная HTML/CSS/JS-версия страницы каталога для презентации руководству.

- широкая таблица с горизонтальной прокруткой;
- закреплённый первый столбец;
- фильтры, поиск, сортировка и выбор товаров;
- адаптивный интерфейс без сборщика.

GitHub Pages может публиковать содержимое ветки `main` напрямую.

## Файлы дизайна

Для следующей замены изображений используйте такие имена:

- `assets/logo/nkt-logo.svg` — логотип в шапке;
- `assets/icons/cart.svg` — белая иконка корзины;
- `assets/images/categories/category-01.jpg` ... `category-12.jpg` — карточки категорий слева направо, сверху вниз;
- `assets/images/products/tf50a-29m29m-01.jpg` — фотография товара.

Для логотипов и иконок предпочтителен SVG. Для фотографий — JPG или WebP без пробелов и кириллицы в имени.
